# かんたんジェノグラム

家族構成を入力して、ジェノグラムを簡単に作成する HTML ツールです。  
介護現場での家族関係整理、説明用のたたき台、デモ表示に使える想定です。

## ファイル構成

- `かんたんジェノグラム.html`
  - 元本です。通常はこちらを編集します。

- `かんたんジェノグラム_デモ版.html`
  - 配布用のデモ版です。
  - 元本の内容を同期しつつ、画面タイトルのみデモ版表記にしています。

- `manifest.json`
  - スマホ・タブレットでホーム画面追加するときの設定です。

- `icon-192.png`
- `icon-512.png`
- `apple-touch-icon.png`
- `icon_app_1024.png`
  - ホーム画面表示用のアイコン画像です。

## 使い方

1. HTML ファイルをブラウザで開きます。
2. 左パネルのテンプレート選択、または家族追加で内容を入力します。
3. 方眼キャンバスへ自動反映されます。
4. 必要に応じて線や囲い、テキストを追加します。
5. `フォルダ保存` で PNG 保存できます。

## スマホ・タブレットで使う場合

- ホーム画面アイコンを正しく表示したい場合は、`file://` で直接開くより、GitHub Pages などの HTTP/HTTPS 経由で開くのがおすすめです。
- iPhone / iPad は Safari で開いて `ホーム画面に追加` を使用してください。
- Android は Chrome 系ブラウザで開いて `ホーム画面に追加` を使用してください。
